# Arturo Juárez Monroy - Portafolio

Curriculum interactivo y portafolio de proyectos construido con React + Vite.

**Hosting:** [https://arturojuarezmonroy.vercel.app](https://arturojuarezmonroy.vercel.app)

---

## Stack

React 19, Vite 7, React Router, Tailwind CSS, GSAP, Three.js, Firebase, Electron.

## Inicio Rápido

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # dist/
npm run preview  # Vista previa del build
```

## Scripts

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Dev server |
| `npm run build` | Build producción |
| `npm run preview` | Preview del build |
| `npm run start` | Abrir en Electron |
| `npm run dist` | Instalador Windows |
| `npm run lint` | ESLint |
